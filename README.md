## Responsive Piano UI
A visually accurate piano keyboard interface built using HTML and CSS.
This project focuses on layout precision, pseudo-elements, and responsive design, creating a realistic UI without JavaScript.

## ✨ Highlights

🎹 Realistic piano key layout

⚫ Black keys created using CSS pseudo-elements

📐 Pixel-precise spacing and alignment

📱 Responsive scaling for smaller screens

🎨 Clean and minimal design

## 🛠️ Tech Stack

HTML5

CSS3

## 🚀 Live Preview

Open index.html in your browser to view the piano layout.
